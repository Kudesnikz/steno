# steno.md

Краткое описанеие приложения которое ты разрабатываешь:
1) Суть приложения - ИИ помощник для сохранения информации об встречах
2) Основной функционал: запись видео звонков в Zoom, Google Meets и тд; составление протоколов записаных встреч с помощью ИИ
3) Десктопное приложение на MacOS (поддержка версий ниже MacOS 12 Monterey не интересует)
4) Приложение написано на Python
5) Обязателен принцип приложения "One Click": Пользователь не должен настраивать драйверы, виртуальные кабели (BlackHole) или выбирать источники вручную. Запись должна работать сразу после запуска
6) Качество видео должно быть достаточным для чтения текста со слайдов, но с минимальным весом файла.
7) Качество аудио: Кристально чистый голос без эха, треска и робо-эффекта (критично для распознавания ИИ).
8) Производительность: Минимальное влияние на систему. Приложение не должно "тормозить" Zoom или перегревать ноутбук.

## Рекомендации

- Использование последних моделей Gemini (gemini-3-flash-preview, gemini-3-pro-preview, gemini-flash-lite-latest)
- Используй genai для работы с Gemini
