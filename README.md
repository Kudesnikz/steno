# Steno - AI Meeting Assistant

**Steno** — это мощное и простое в использовании приложение для macOS, предназначенное для записи встреч и автоматического создания протоколов с использованием искусственного интеллекта (Google Gemini).

Приложение записывает видео экрана и звук (системный + микрофон), а затем использует нейросеть для анализа записи и генерации структурированного отчета, включающего саммари, принятые решения и план действий.

## Возможности

*   **Запись в один клик:** Не требует сложной настройки драйверов или виртуальных кабелей.
*   **Двойная запись:** Записывает видео экрана и отдельную дорожку с микрофона для максимальной точности распознавания речи.
*   **ИИ-протоколирование:** Автоматически создает подробные протоколы встреч (Markdown).
*   **Нативная интеграция:** Работает из строки меню macOS, потребляет минимум ресурсов.
*   **Поддержка моделей Gemini:** Использует новейшие модели (Gemini).

## Инструкция по использованию

### 1. Первый запуск и Разрешения

При первом запуске macOS может запросить разрешения. Для корректной работы приложения необходимо предоставить:
*   **Запись экрана (Screen Recording):** Чтобы видеть презентации и участников.
*   **Микрофон (Microphone):** Чтобы слышать ваш голос.
*   **Уведомления (Notifications):** Чтобы сообщать о завершении обработки.

*Если разрешения не были предоставлены, перейдите в `System Settings -> Privacy & Security` и включите их для приложения Steno вручную.*

### 2. Настройка перед началом

1.  Нажмите на иконку приложения в строке меню.
2.  Перейдите в **Settings -> Set API Key...**.
3.  Введите ваш API ключ Google Gemini (получить можно в [Google AI Studio](https://aistudio.google.com/)).
4.  (Опционально) В **Settings -> Video Quality** выберите качество записи (по умолчанию Medium).
5.  (Опционально) В **Settings -> AI Model** выберите желаемую модель.

### 3. Запись встречи

1.  Нажмите **Start Recording** в меню.
2.  Проводите встречу как обычно.
3.  По завершении нажмите **Stop** в меню.
4.  Запись будет сохранена локально.

### 4. Создание протокола (AI)

1.  Откройте меню **Recent Recordings**.
2.  Выберите файл встречи, который хотите обработать.
3.  Иконка изменится, показывая процесс обработки.
4.  Приложение загрузит файлы, проанализирует их и сохранит протокол (`_protocol.txt`) в той же папке.
5.  По завершении придет уведомление. Вы можете открыть протокол через меню **Recent Protocols**.

## Сборка приложения (для разработчиков)

Для сборки приложения из исходного кода вам понадобится Python 3.13 (рекомендуется) и установленные зависимости.

### Требования

1.  **Python 3**
2.  **FFmpeg:** Исполняемый файл `ffmpeg` должен находиться в папке `bin/` в корне проекта.

### Установка зависимостей

```bash
pip install -r requirements.txt
# Если файла requirements.txt нет, установите основные пакеты:
pip install rumps google-genai pyobjc certifi py2app
```

### Сборка .app бандла

Используется `py2app` для создания standalone приложения.

```bash
python3 setup.py py2app
```

После успешной сборки приложение будет находиться в папке `dist/Steno.app`.

## Создание .dmg файла

### Требования

*   Утилита `create-dmg` (можно установить через Homebrew):
    ```bash
    brew install create-dmg
    ```

### Инструкция

1.  Убедитесь, что приложение собрано (`dist/Steno.app` существует).
2.  Запустите скрипт сборки:
    ```bash
    ./build_dmg.sh
    ```
3.  Готовый образ `Steno.dmg` появится в корневой папке проекта.
